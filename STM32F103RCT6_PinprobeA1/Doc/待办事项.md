## RS485 当前状态

- **协议**：Modbus RTU，9600bps，8N1，站号 1
- **TX**：`HAL_Delay(10)` + `HAL_UART_Transmit(100ms超时)` + `HAL_Delay(10)`
- **RX**：USART3 DMA 双缓冲 + IDLE 中断，`IO_Read` 直接读 `usart3_buff_IsReady`，CRC 校验，最多重试 5 次
- **方向控制**：硬件自动换向芯片（MAX13487 或同类），无需软件控制

## 关键定时参数

| 参数 | 值(ms) | 说明 |
|------|--------|------|
| `LOCK_PRESS_MS` | 300 | 锁定按钮需保持 300ms |
| `LOCK_IDLE_MS` | 1000 | 锁定按钮冷却 1s |
| `DOOR_CLOSE_CONFIRM_MS` | 700 | 双按钮同时按 700ms 关门 |
| `DOOR_OPEN_CONFIRM_MS` | 300 | 按钮按 300ms 开门 |
| `RELEASE_DELAY_MS` | 200 | 按钮释放 200ms |
| `AIR_STABILIZE_MS` | 3000 | 关门后气压稳定等 3s |
| `AIR_CHECK_INTERVAL_MS` | 1500 | 气压检测间隔 1.5s |
| `RS485_TX_TIMEOUT_MS` | 100 | TX 超时保护 |
| IWDG 超时 | ~4000 | 看门狗 4s |
| WatchDog 喂狗间隔 | 500 | 独立线程 |
